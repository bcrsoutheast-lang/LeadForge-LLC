How LeadForge works

The site is built around a simple exclusive request model that keeps the process cleaner for both homeowners and contractors.

1

Homeowner chooses a service

The homeowner starts by selecting the service they need and viewing matching contractors.

2

One direct request is sent

The homeowner picks one contractor and submits one exclusive request for that project.

3

Contractor unlocks the opportunity

The contractor sees the request, unlocks it, and gets the homeowner’s full contact details.

Ready to test the whole flow?

Start from the homeowner side first, then run the contractor submit and admin approval flow. That will make it easy to spot exactly which button or page is failing.
